## ‚öôÔ∏è Configuration Guidelines

### Core Settings

- **Custom Domain**  
  You must bind your application to a custom domain. Ensure you enable the gRPC function in the **Network menu** under Cloudflare settings.

- **Node Import Instructions**  
  - **Shadowsocks**: You cannot import directly; you need to manually configure it after checking the generated node information.
  - **Socks5**: Imported information might be incomplete; manual adjustments could be necessary.

### Default Passwords & Security

- **Initial Password**  
  When you first open the app, it will prompt you to set a password. This helps secure your settings and generate dynamic UUID or TROJAN passwords. Ensure to set your KV name correctly to store your configurations.

- **Access Password**  
  After your first login, go to the edit configuration page to set a password. This protects your node settings from unauthorized changes.
